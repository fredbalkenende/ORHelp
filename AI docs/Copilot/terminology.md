# OfficeReports Terminology

This file defines common terms and synonyms used in OfficeReports help content.  
The purpose is to help match different words that users may use for the same concept.

## General Rule

Users may describe the same feature or concept with different words.  
When interpreting help questions, treat the terms below as closely related when the context matches.


## Survey Data Terms

### Case

A **case** usually means one record in the data.  
A case is often also called a **respondent**.

**Related terms:**

* case
* cases
* respondent
* respondents
* interview
* interviews

**Example:**  
"How many cases are in this file?" means the same as "How many respondents are in this file?"

### Respondent

A **respondent** usually means a person or unit represented by one data record.  
A respondent is often also called a **case**.

**Related terms:**

* respondent
* respondents
* case
* cases

### Variable

A **variable** is a field or column in the data.  
Depending on context, a variable may also be called a **question**.

**Related terms:**

* variable
* variables
* question
* questions
* field
* fields
* column
* columns

**Note:**  
Not every question is always stored as one simple variable, but in many help questions these terms are used interchangeably.

### Value Labels

**Value labels** are the text descriptions connected to coded values.

**Related terms:**

* value labels
* answer labels
* category labels
* labels
* codes and labels


## Table and Analysis Terms

### Crosstab

A **crosstab** is a table showing results by one or more background variables or segments.  
A crosstab may also be called a **cross tab**, **cross tabulation**, or **banner table**.

**Related terms:**

* crosstab
* crosstabs
* cross tab
* cross tabs
* cross tabulation
* cross tabulations
* banner table
* banner tables
* tabulation
* tables

**Example:**  
"Create a banner table" may mean the same as "Create a crosstab."

### Banner

A **banner** is the set of columns or breaks used in a crosstab or banner table.

**Related terms:**

* banner
* banners
* break
* breaks
* column break
* banner definition

### Banner Table

A **banner table** is another term for a **crosstab**.

**Related terms:**

* banner table
* banner tables
* crosstab
* crosstabs
* cross tab
* cross tabulation

### Stub

A **stub** is the row definition or row variable area of a table.

**Related terms:**

* stub
* stubs
* row definition
* row area
* table rows

### Base

The **base** is the number of valid cases or respondents behind a result.

**Related terms:**

* base
* bases
* sample base
* respondent base
* unweighted base
* weighted base
* n

**Example:**  
"What is the base?" may mean "How many respondents are included in this result?"

### Net

A **net** is a combined result made from multiple answer categories.

**Related terms:**

* net
* nets
* netted result
* combined category
* grouped result

### Mean

A **mean** is the average value.

**Related terms:**

* mean
* average
* arithmetic mean

### Percentage

A **percentage** is the share of respondents or cases in a category.

**Related terms:**

* percentage
* percent
* share
* proportion

### Significance Test

A **significance test** checks whether differences are statistically meaningful.  
It may also be called a **sig test** or **statistical significance test**.

**Related terms:**

* significance test
* significance testing
* sig test
* sig testing
* statistical test
* statistical significance
* significance letters


## Reporting and Output Terms

### Populate

To **populate** means to fill a PowerPoint shape, table, chart, or other report object with data.

**Related terms:**

* populate
* fill
* insert data into
* update object
* send to PowerPoint

### Linked Object

A **linked object** is a PowerPoint object connected to data in Excel or another OfficeReports source.

**Related terms:**

* linked object
* link
* linked shape
* linked chart
* linked table
* linked content

### Refresh

To **refresh** means to update an existing linked result using the latest available data.

**Related terms:**

* refresh
* update
* reload
* repopulate
* rerun

**Note:**  
In some contexts, **refresh** means updating an existing object, while **populate** means filling a new or selected object.

### Template

A **template** is a predefined report structure or slide setup used for repeated reporting.

**Related terms:**

* template
* reporting template
* slide template

## Practical Interpretation Rules

### Equivalent Terms Often Used by Users

Treat these as closely related when context matches:

* case = respondent
* crosstab = cross tab = cross tabulation = banner table
* significance test = sig test
* variable = question
* workbook = Excel file
* worksheet = sheet
* refresh = update
* populate = fill with data
* presentation = slide deck

### Terms That May Differ by Context

The following terms are related but not always identical:

* variable and question
* refresh and populate
* base and total
* chart and graph
* table and crosstab

Use surrounding context to decide the intended meaning.

## Purpose of This File

This file is intended to improve help retrieval and interpretation of user questions by mapping common synonyms and alternate terminology used in market research, Excel, and PowerPoint reporting workflows.
