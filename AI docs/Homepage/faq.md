# OfficeReports FAQ

This FAQ answers common questions about the three OfficeReports products for market research teams: Presento, Calculo, and Intelligo.

## Quick product guide

### What is OfficeReports?

OfficeReports provides tools for survey analysis and reporting automation in Microsoft Office.

### Which OfficeReports product should I use?

Choose **Presento** if you already have results in Excel and want to automate PowerPoint reporting.

Choose **Calculo** if you need to analyze survey data and create crosstabs in Excel.

Choose **Intelligo** if you need both survey analysis and automated PowerPoint reporting.

### What is the difference between Presento, Calculo, and Intelligo?

The easiest way to think about it is this:

* **Presento** starts after the analysis is ready and focuses on PowerPoint automation.
* **Calculo** focuses on survey analysis and table creation in Excel.
* **Intelligo** covers both analysis and reporting automation.

### Which product includes both analysis and reporting automation?

**Intelligo**.

## Presento FAQ

### What does Presento do?

Presento links Excel content to PowerPoint and can populate shapes, tables, and charts in PowerPoint. When Excel data changes, the PowerPoint presentation can be updated.

### Is Presento for PowerPoint automation?

Yes. Presento is designed for PowerPoint report automation.

### Can Presento update PowerPoint charts and tables automatically?

Yes. Presento can update linked PowerPoint content when the source Excel data changes.

### Does Presento create native PowerPoint slides?

Yes. The result is a native PowerPoint presentation that can still be edited in PowerPoint.

### Can Presento analyze survey data?

Not as its main purpose. Presento is designed for PowerPoint reporting automation once the results already exist. If you need to create crosstabs or analyze raw survey data, Calculo or Intelligo is the better fit.

### When should I use Presento?

Use Presento when your main need is recurring PowerPoint reporting based on Excel data.

## Calculo FAQ

### What does Calculo do?

Calculo is used to analyze survey data in Excel. It supports cross tabulation, percentages, means, nets, and significance testing.

### Is Calculo a crosstab tool?

Yes. Calculo is designed for cross tabulation and survey data analysis in Excel.

### Can Calculo perform significance testing?

Yes. Calculo supports significance testing used in market research reporting.

### Can Calculo help me analyze survey data without SPSS syntax?

Yes. Calculo is designed for users who want survey analysis in Excel without depending on complex scripting.

### Can Calculo automate PowerPoint reporting?

Calculo is designed for analysis in Excel, not for automated PowerPoint population. If you also need to automate PowerPoint deliverables, Intelligo is the better fit.

### When should I use Calculo?

Use Calculo when your main need is survey data analysis and cross tabulation in Excel.

## Intelligo FAQ

### What does Intelligo do?

Intelligo combines survey data analysis and PowerPoint reporting automation. It includes all core Calculo functionality and adds PowerPoint integration.

### Does Intelligo include Calculo functionality?

Yes. Intelligo covers the same type of survey analysis workflow as Calculo and adds PowerPoint reporting automation for teams that need both.

### Can Intelligo create cross tabulations?

Yes. Intelligo supports cross tabulation and survey data analysis.

### Can Intelligo automate PowerPoint reports?

Yes. Intelligo supports PowerPoint reporting automation.

### Is Intelligo the best choice for end-to-end survey reporting?

Yes. Intelligo is the best fit when you need both analysis and automated presentation output.

### When should I use Intelligo instead of Calculo?

Use Intelligo instead of Calculo when you need both survey analysis and PowerPoint automation.

## Use Case FAQ

### I need to update recurring PowerPoint reports from Excel. Which product should I use?

Use **Presento** if you only need PowerPoint automation.  
Use **Intelligo** if you also need survey analysis and cross tabulation.

### I need to create crosstabs in Excel. Which product should I use?

Use **Calculo** or **Intelligo**.

### I need both data analysis and presentation automation. Which product should I use?

Use **Intelligo**.

### I already analyze data in Excel and only need to automate PowerPoint. Which product should I use?

Use **Presento**.

### I want to replace a workflow that depends on manual Excel analysis and manual PowerPoint updates. Which product should I use?

Use **Intelligo** if you want one workflow covering both analysis and reporting.  
Use **Presento** if the analysis part is already handled and only reporting needs automation.

## Capability FAQ

### Does OfficeReports work with Excel?

Yes. Excel plays an important role in all three products, but in different ways:
* **Presento** uses Excel as the source for PowerPoint updates
* **Calculo** uses Excel as the environment for survey analysis
* **Intelligo** uses Excel for analysis and connects that workflow to PowerPoint reporting

### Does OfficeReports work with PowerPoint?

Yes, but not in the same way across all products:
* **Presento** is focused on PowerPoint automation
* **Intelligo** also supports PowerPoint reporting automation
* **Calculo** is focused on analysis in Excel rather than PowerPoint output

### Can OfficeReports automate market research reporting?

Yes.
* **Presento** automates PowerPoint reporting when the analysis is already available
* **Intelligo** combines analysis and reporting automation in one workflow

## Comparison FAQ

### What is the main difference between Calculo and Intelligo?

Calculo is for survey analysis in Excel. Intelligo includes that analysis functionality and also automates PowerPoint reporting.

### What is the main difference between Presento and Calculo?

Presento focuses on PowerPoint automation. Calculo focuses on survey data analysis in Excel.

### What is the main difference between Presento and Intelligo?

Presento focuses on reporting automation. Intelligo combines reporting automation with survey analysis.

### Which OfficeReports product is the most complete?

Intelligo offers the broadest workflow coverage because it supports both survey analysis and PowerPoint reporting automation.

### Why would I choose Presento instead of Intelligo?

Choose Presento when your analysis is already handled and your only goal is to automate PowerPoint reporting. Choose Intelligo when you need both analysis and reporting automation in the same solution.

### Why would I choose Calculo instead of Intelligo?

Choose Calculo when your work is focused on survey analysis and crosstabs in Excel, and you do not need automated PowerPoint reporting.
