# OfficeReports Product Comparison

This document explains the difference between the three OfficeReports products for market research reporting and analysis: Presento, Calculo, and Intelligo.

## Quick Summary

* **Presento** is used to link Excel data to PowerPoint and update slides automatically.
* **Calculo** is used to analyze survey data in Excel and create cross tabulations and significance tests.
* **Intelligo** combines **Calculo** and **Presento**. It supports both survey data analysis and automated PowerPoint reporting.

## Which product should I choose?

### Choose Presento if you want to:

* link Excel ranges to PowerPoint charts, tables, and text
* update PowerPoint reports automatically when Excel data changes
* create native PowerPoint presentations that remain editable
* automate recurring report production in PowerPoint

### Choose Calculo if you want to:

* analyze survey data in Excel
* create cross tabulations
* calculate percentages, means, nets, significance tests, and statistical summaries
* work with market research data without scripting

### Choose Intelligo if you want to:

* analyze survey data and automate report creation in one solution
* create cross tabulations in Excel
* send results directly into PowerPoint
* automate recurring survey reporting from data to presentation

## Product Details

## Presento

**Main purpose:** PowerPoint report automation

Presento links content from Excel to PowerPoint. It can populate PowerPoint shapes, tables, and charts with data from Excel. When the source data changes, the presentation can be updated automatically.

Presento is best for users who already have their analysis in Excel and want to automate PowerPoint reporting.

### Presento is typically used for:

* recurring tracker reports
* brand health reports
* monthly or quarterly management presentations
* client reporting in PowerPoint
* updating charts and tables from Excel

### Presento does not focus on:

* survey data analysis
* cross tabulation
* significance testing
* importing and analyzing raw survey datasets

## Calculo

**Main purpose:** Survey data analysis in Excel

Calculo is designed for market researchers who need to analyze survey data in Excel. It supports cross tabulation and statistical analysis without requiring SPSS syntax or advanced scripting.

Calculo is best for users who want to turn raw survey data into tables and results inside Excel.

### Calculo is typically used for:

* cross tabulations
* banner analysis
* significance testing
* percentages, counts, means, and nets
* survey data processing in Excel

### Calculo does not focus on:

* PowerPoint slide automation
* direct report population in PowerPoint
* end-to-end presentation generation

## Intelligo

**Main purpose:** Survey analysis plus PowerPoint reporting automation

Intelligo includes all Calculo functionality and adds integration with PowerPoint for automated reporting. It is the most complete solution for teams that want to go from survey data to presentation output in one workflow.

Intelligo is best for users who need both analysis and reporting automation.

### Intelligo is typically used for:

* analyzing survey data
* creating cross tabulations
* significance testing
* automating PowerPoint reporting
* recurring multi-country or multi-wave reporting
* producing presentation-ready outputs faster

### Intelligo includes:

* all core analysis capabilities of Calculo
* integration with PowerPoint
* support for report automation workflows

## Common Questions

### What is the difference between Calculo and Intelligo?

Calculo focuses on survey data analysis in Excel. Intelligo includes the same analysis functionality and also supports PowerPoint reporting automation.

### What is the difference between Presento and Intelligo?

Presento focuses on PowerPoint automation using Excel as the source. Intelligo also supports survey data analysis and cross tabulation.

### Can I use Presento for survey data analysis?

Presento is mainly for report automation in PowerPoint. For survey data analysis and cross tabulation, Calculo or Intelligo is the better choice.

### Can I use Calculo to automate PowerPoint slides?

Calculo is focused on analysis in Excel. If you need PowerPoint automation as well, Intelligo is the better fit.

### Which product is the most complete?

Intelligo is the most complete product because it combines survey analysis and PowerPoint reporting automation.
