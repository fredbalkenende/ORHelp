# Tables and Verbatim Lists

Use **Tables** and **Verbatim Lists** to add tables and lists to your workbook. You can also use **Bulk Edit** to update multiple tables at the same time.

![Create Tables and Lists from the OfficeReports Ribbon Tab](/Resources/Images/TablesTextsLists.png)

## Tables

OfficeReports tables are Excel named ranges. This means you can add rows and columns in the worksheet and OfficeReports still recognizes the tables. You can also move tables by using cut and paste.

To insert one or more tables:

1. Click **Table**.
2. Select the option you want.

You can also create tables by dragging variables from the Data Editor in the OfficeReports App.

Read more about [Tables](tables.md).

## Verbatim Lists

Use **Verbatim Lists** to insert a list of open-ended answers. You can also list results from other variables. For example, you can create a list of comments sorted by the responses to another question.

To insert a Verbatim List:

1. Click **Text-List**.
2. Define the settings for the list.

Read more about [Verbatim Lists](verbatim-lists.md).

## Mass Update

Use **Bulk Edit** when you need to update multiple tables with the same settings in one process.

![Bulk Edit Form](/Resources/Images/MassUpdate_1009x1032.png)

### Select Tables

You can select tables in three ways:

1. Select the tables directly in the list.
2. Select worksheets.
3. Select banners.

You can combine worksheet selection and banner selection.

### Edit Settings

1. Click one of the **Edit** buttons, for example **Filter**.
2. Define the setting.
3. Click **OK**.

When a setting is defined, the button turns green.

To remove a defined setting:

1. Click the green button.
2. Click **Delete definition** in the window that opens.
3. Click **OK**.

After you click **OK**, OfficeReports applies the settings behind the green buttons to all selected tables.
