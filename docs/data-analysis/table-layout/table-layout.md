# Table Layout

Use **Table Layout** to define the default table layouts for OfficeReports Crosstabs and Tables.

OfficeReports uses different layouts depending on where you work. In Excel, the default layout typically shows count, percentage, and significance characters. In PowerPoint, the default layout is usually more compact. It can hide count, show significance by using colors, and separate totals from categories. This makes it easier to populate charts and tables in PowerPoint.

Table layouts consist of table area settings, table border settings, and general layout properties.

## Default Table Layout

![Table Layout Settings](/Resources/Images/TableLayout.png)

In **Table Layout**, you can:

1. Set the default layout for Excel.
2. Set the default layout for PowerPoint.
3. Create a new table layout.
4. Edit an existing table layout.
5. Change significance colors.
6. Change the standard texts used in tables.

> **Important:** The significance colors are shared across reports. If you change them, the new colors are also used when you refresh older PowerPoint presentations and Excel workbooks.

## Report Table Layout

You can override the default table layout for a specific workbook or presentation. The significance colors are the exception and remain shared.

To set the layout for a specific report:

1. Open the OfficeReports ribbon tab.
2. Click **Table Layout**.

![Table Layout Settings in the OfficeReports Ribbon Tab](/Resources/Images/TableLayoutInRibbon.png)
