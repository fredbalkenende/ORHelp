# Edit Table Layout - Properties

When you select a table layout, OfficeReports shows a sample table based on example data to the right of the **OfficeReports Layout** pane.

![Table Layout definition settings](/Resources/Images/OfficeReports Analytics/Reverse Significance Test.png)

## Default Size

Use **Default size** to define the default row height and column width in the table layout.

## Reverse Significance Test

Use **Reverse Significance Test** when higher values represent a worse result.

By default, OfficeReports uses a green fill color when a value is significantly higher in a statistical test shown with fill colors in Excel. For some variables, a higher value means a worse result. In that case, use an alternative table layout where a significantly higher value is shown with a different fill color, for example red.
