# Bulk Create Tables

Use **Bulk Create Tables** to create multiple tables in one process. By setting the correct options at report level and variable level, you can generate tables with different settings in a single run.

![Create Many Tables in the OfficeReports Ribbon Tab](/Resources/Images/CreateManyTablesRibbon_529x469.png)

## Default Settings

Before you create many tables, define the required settings for the report and for the variables you want to use.

![Create Many tables Form](/Resources/Images/Create Many Tables_838x790.png)

## Ordinal Variables

Define variables as **Ordinal Variables** when you want OfficeReports to use specific statistics for the tables created from those variables.

![Options for Ordinal variables](/Resources/Images/Ordinal Variables.png)
