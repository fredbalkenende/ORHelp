# Tables

Use tables to present your data in Excel workbooks or PowerPoint presentations. You can create tables from the **Tables** menu in the OfficeReports ribbon tab or by dragging variables from the Data Editor.

![Tables menu in the OfficeReports Ribbon Tab](/Resources/Images/TableChartsInRibbon.png)

1. Create a table from the **Tables** menu in the OfficeReports ribbon tab, or drag variables from the Data Editor to the worksheet.
2. Double-click the table in the OfficeReports App to edit it.

OfficeReports supports standard frequency tables and cross tables. If you want to show multiple variables in the same table by using the same banner, create a grid table.

![Crosstab with means](/Resources/Images/GridTable_905x192.png)

OfficeReports uses either the selected **PowerPoint** table layout or the selected **Excel** table layout, depending on whether your output is for PowerPoint or Excel. Define these defaults in **Layout Settings**, which opens when you click **Table Layout** in the OfficeReports ribbon tab.

Read more about [Creating Tables](create-table.md) and [Table Layout](table-layout/table-layout.md).
