# Installation problems

OfficeReports can be downloaded and installed from our [homepage](https://www.officereports.com/download). The installation uses Microsoft ClickOnce, and most users can install OfficeReports without contacting their IT department.

1. Go to the [homepage](https://www.officereports.com/download).
2. Download and install OfficeReports.
3. Continue the installation without Administrator rights.

You do not need Administrator rights to install OfficeReports.

If you run into problems, [contact us](https://www.officereports.com/contact-us) immediately.
