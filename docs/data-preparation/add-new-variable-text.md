# Add new Variable Texts

You can replace existing variable texts in an OfficeReports file in one process. Import an Excel file that contains the variable names and the new variable texts.

1. Open **Add Data**.
2. Select **Variable Texts**.
3. Select the Excel file to import.
4. Make sure Column 1 contains **Variable Names**.
5. Make sure Column 2 contains **Variable Texts**.
6. Import the file.

During the import, OfficeReports replaces all existing variable texts with the new variable texts.
