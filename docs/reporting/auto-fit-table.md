# AutoFit

Use **AutoFit** to make a PowerPoint table match the number of rows and columns in the data range.

When you populate a PowerPoint table, **AutoFit** adds or removes rows and columns as needed so the table fits the full data range. This also works when the range does not start in the first cell of the PowerPoint table.

## Preserve Width and Height

Use **Preserve the current table width and height** to keep the table size unchanged.

Without this option, the table becomes larger when rows or columns are added and smaller when rows or columns are removed.

## Benefits

**AutoFit** helps you:

- avoid manually resizing the table
- update the table automatically when the data range changes, for example in a pivot table

## Notes

> **Note:** Do not use **AutoFit** when you want the table to grow, for example when you add a new month in the next monthly report.

> **Note:** Do not use **AutoFit** when you want to populate the PowerPoint table with multiple data ranges.
