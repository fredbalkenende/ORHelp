# Sticky Colors

Use **Sticky Colors** to keep chart colors consistent based on text values in the workbook.

Before OfficeReports can apply sticky colors automatically, you must define which colors to use for specific texts.

![Sticky Colors in the Presento Ribbon Tab](/Resources/Images/StickyColors in ribbon tab.png)

## Create a Sticky Color Definition

Click **Sticky Colors** in the ribbon tab to open the definition window.

![Sticky Colors Form](/Resources/Images/StickyColors Form.png)

To define a sticky color:

1. Select a cell in the worksheet that contains the text you want to use.
2. Click **Use Cell Text**.
3. Select a cell in the worksheet that contains the fill color you want to use.
4. Click **Use Cell Color** or **Use Font Color**.
5. Select a cell in the worksheet that contains the font color you want to use.
6. Click **Use Cell Color** or **Use Font Color**.

## Apply Sticky Colors

After you create the sticky color definitions:

1. Select the chart in PowerPoint.
2. Open the **Populated Pane** on the right side.
3. Select **Use Sticky Colors**.

The chart then uses the defined colors based on the texts in the chart.

![Sticky Colors in the Populate Pane](/Resources/Images/StickyColors in Pane.png)
