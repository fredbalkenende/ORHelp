# Verbatim Lists

Use **Verbatim Lists** to report open-ended answers in your workbook.

A Verbatim List can include the open-ended text together with one or more additional variables. You can also filter the list and sort it by the first variable.

## Create a Verbatim List

1. Select an empty cell in the worksheet.
2. Make sure there is enough empty space around the cell for the Verbatim List.
3. Click **Verbatim List** in the OfficeReports ribbon tab.

You can also right-click the empty cell and select **Create Verbatim List**.

## Edit a Verbatim List

Double-click a cell in the Verbatim List and the Verbatim List Form will popup.
