# Data Visualization

Use data visualization in PowerPoint to link workbook data to shapes, tables, and charts in your presentation.

## Attach a Workbook

Before you can work with data visualization, attach a workbook to the presentation.

If you work with raw data, OfficeReports creates and attaches a workbook automatically when you start the OfficeReports app and click **New**.

![Attach a Workbook user interface](/Resources/Images/NewPPTSettings.png)

By default, OfficeReports writes data directly into the charts. Change this setting if you want to edit the chart data in the chart workbook.

By default, OfficeReports also creates and attaches a worksheet to each slide you work with. This is important if you want to improve productivity and use **Copy Slide (Setup)**. Change this setting if you want to control and organize the worksheets yourself.

## Reorganize the Data

If you use raw data, create crosstabs first.

If you use another data source, you may need to reorganize the data. Read more about [Lookup Tables, Lookup Ranges and Linked Ranges](../data-preparation/reorganize-data.md). You can also read more about [handling crosstabs created using other tools](../data-preparation/external-crosstabs.md).

## Populate and Link

The attached workbook is shown in the **Workbook Pane** in PowerPoint.

You can click the

![popout icon](/Resources/Images/popout-icon_32x32.png)

icon to open the workbook in a floating window. The functionality in the workbook window is the same as in the **Workbook Pane**.

You can click the

![Excel icon](/Resources/Images/openinexcelicon_32x30.png)

icon in the workbook window to open the workbook in Excel.

### Use the Workbook Pane or Workbook Window

1. Select the shape, table, or chart on the slide.
2. Select a data range in the worksheet.
3. Right-click the selected range.

Read more about [Populate Charts](populate-chart.md), [Populate Tables](populate-table.md), and [Populate Shapes](populate-shape.md).

### Use Excel

1. Select a range in Excel.
2. Click **Populate and Link** in the OfficeReports ribbon tab in PowerPoint.
