# Refresh Report

Use **Refresh Report** to update all tables, charts, and shapes in a report.

![Refresh report in the OfficeReports Ribbon Tab](/Resources/Images/RefreshInRibbon.png)

Use **Refresh Report** when changes affect multiple OfficeReports objects in the report.

Examples include:

- adding new data
- changing variable texts or category names
- adjusting formulas for derived variables
- changing report filters
