# Interactive Filters

Use **Interactive Filters** in Slide Show mode to filter the data shown on a slide. This helps you engage your audience without creating as many slides.

## Define Interactive Filters

You can add dropdowns and slicers to your slides.

After you add an interactive filter, you can edit it from the **Interactive Filter** ribbon tab.

The interactive filter is a group of shapes. You can edit the shapes, for example by changing colors, borders, and size.

> **Important:** Do not delete or rename any of the shapes in the interactive filter.

## Use Interactive Filters

In Slide Show mode, click the dropdown and slicer shapes to filter the data shown on the slide.
