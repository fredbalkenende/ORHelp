# AI Assistant in PowerPoint

Use the AI Assistant in PowerPoint to generate content based on your presentation context. The AI Assistant is powered by OpenAI's GPT-4.

![Ai Assitant in the Presento Ribbon Tab](/Resources/Images/AIAssistant.png)

## Settings

To get the best results from the AI Assistant, provide information about the presentation context.

## Context

Describe what the presentation is about. Also specify who the audience is and what style you want the answers to use.

## Use Your Own Key

If you have your own OpenAI subscription, OfficeReports recommends that you use your own API key.

1. Go to [OpenAI API keys](https://platform.openai.com/api-keys).
2. Create an API key.
3. Open the AI Assistant settings.
4. Click the **API Key** tab.
5. Enter the API key.

Read more about getting your own API key [here](get-openai-api-key.md).

## Privacy and Security

OfficeReports communicates with GPT-4 by using the OpenAI API. OpenAI states that API inputs and outputs may be securely retained for up to 30 days to identify abuse. OpenAI does not use this API communication to train the AI models.

Read more about [privacy and security](https://openai.com/enterprise-privacy/) when using the API.
