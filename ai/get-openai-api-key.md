# Get an OpenAI API Key

Use your own OpenAI API key to connect the AI Assistant to your OpenAI account.

1. Sign up for the OpenAI API.
2. Open your account and go to [View API keys](https://platform.openai.com/account/api-keys).
3. Click **Generate new secret key**.
4. Copy the API key.
5. Open **AI Assistant > Settings** in OfficeReports.
6. Paste the API key.

You can now use the AI Assistant with your own OpenAI API key.
